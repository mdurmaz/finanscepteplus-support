# FinansCepte+ Destek Sayfası

Bu repository GitHub Pages ile host edilen destek sayfası içindir.

## Kurulum

1. Bu repository'yi GitHub'da oluşturun (public olmalı)
2. Repository Settings > Pages bölümüne gidin
3. Source olarak "main" branch'ini seçin
4. Save butonuna tıklayın

## URL

Sayfanız şu formatta olacak:
- `https://[kullanıcı-adınız].github.io/[repo-adı]/`

Örnek:
- `https://finanscepte.github.io/finanscepteplus-support/`

## Dosyalar

- `index.html` - Ana destek sayfası
- `privacy.html` - Gizlilik politikası sayfası

## Özelleştirme

- E-posta adresini değiştirmek için `index.html` dosyasındaki e-posta adresini güncelleyin
- Renkleri değiştirmek için CSS'deki renk kodlarını düzenleyin
- İçeriği ihtiyacınıza göre özelleştirin
